# EclipseChase.is

A PWA that helps visitors to Iceland find the clearest skies for the **total solar eclipse on August 12, 2026** — real-time weather, eclipse geometry, horizon checks, and a 5-profile recommendation engine for picking the best viewing spot.

Live: <https://eclipsechase.is>

**Stack:** Nuxt 4 + TypeScript + TailwindCSS + Supabase + Mapbox GL, deployed on Vercel.

## Quickstart

```bash
npm install
npm run dev
```

Dev server runs on `http://localhost:3000`.
